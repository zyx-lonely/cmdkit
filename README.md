# 🐧 Linux 命令工具箱

专为 Linux 新手打造的桌面命令速查 + 系统诊断工具。

## 功能

### 📖 命令速查
- **130+ 常用 Linux 命令**，按 13 个分类浏览
- **难度分级**：⭐入门 / ⭐⭐进阶 / ⭐⭐⭐高级
- **中文搜索**：输入"查看进程"、"安装软件"等中文也能找到命令
- **场景引导**：每个命令标注典型使用场景
- **关联推荐**：查看一个命令时推荐相关命令

### ⚡ 命令执行
- 内置终端模拟器，一键执行命令
- **超时保护**（60 秒自动停止）
- **取消按钮**：可随时取消正在执行的命令
- **输出高亮**：错误红色、警告黄色、成功绿色
- **危险命令警告**：自动检测并确认 `rm -rf` 等破坏性命令
- 执行历史记录（支持方向键切换）

### 🖥️ 系统信息
- 操作系统、内核、CPU、内存、磁盘信息
- 实时监控（CPU / 内存 / 磁盘，3 秒刷新）

### 🐳 Docker 管理
- 列出所有容器（含状态、端口）
- 启动 / 停止 / 重启容器
- 查看容器日志

### 🔗 SSH 管理
- 添加 / 编辑 / 删除 SSH 连接
- 一键测试连接

### 📦 安装指南
- AI 工具（Ollama、CUDA、PyTorch 等）
- 开发环境（Miniconda、Node.js、Go、Docker 等）
- 常见问题（pip 慢、端口占用、磁盘空间不足等）

### ⭐ 收藏 & 笔记
- 收藏常用命令，快速访问
- 为每个命令添加个人笔记

### 🎨 自定义
- 深色 / 浅色主题
- 12 种强调色
- 3 种字体大小
- 自定义命令别名
- 数据导出

## 技术栈

| 层 | 技术 |
|---|---|
| 桌面框架 | [Wails v2](https://wails.io/) |
| 后端 | Go 1.23 |
| 前端 | 原生 JavaScript + Vite |
| 样式 | 纯 CSS 变量 |

## 构建

```bash
# 开发模式
wails dev

# 构建
wails build
```

## 项目结构

```
├── app.go              # Go 后端（命令数据、系统接口、Docker/SSH）
├── main.go             # Wails 入口
├── frontend/
│   ├── index.html      # 主页面
│   ├── src/
│   │   ├── main.js     # 前端逻辑
│   │   └── style.css   # 样式表
│   └── dist/           # 构建产物
└── build/              # 打包配置
```
