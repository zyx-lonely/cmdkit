# 🐧 Linux 命令工具箱

专为 Linux 新手打造的桌面命令速查 + 系统诊断工具。

## v1.1.0 功能（最新）

### 📥 配置导入
- 从文件中导入 JSON 配置（收藏、笔记、SSH、别名、执行历史、收藏分类）
- 支持选择 `cmdkit-config.json` 或任意兼容格式文件

### 🔍 搜索难度筛选
- 搜索框下方增加过滤标签：⭐入门 / ⭐⭐进阶 / ⭐⭐⭐高级
- 点击筛选仅显示对应难度级别的命令

### ⌨️ 命令面板（Ctrl+P）
- **Ctrl+P / Ctrl+K** 弹出全局命令搜索面板
- 实时过滤 130+ 命令，Enter 直接执行选中命令
- 全键盘操作，无需鼠标

### 🏷️ 收藏分类管理
- 右键点击收藏 ⭐ 可为收藏命令添加分类标签
- 收藏视图中显示分类过滤条，按分类快速定位
- 自动列出所有已使用的分类

### 📊 命令对比模式
- 打开比较弹窗，从下拉列表选择两个命令
- 并排显示名称、描述、语法、难度、场景、示例

### 🌗 自动主题（跟随系统）
- 监听系统 `prefers-color-scheme` 自动切换深色/浅色
- 用户手动设置主题后自动失效，尊重用户选择

### ⚙️ 进程管理器
- 查看 Top 50 进程（按 CPU 排序）：PID、CPU%、MEM%、用户、内存占用
- 一键结束进程
- 侧边栏新增进程管理入口按钮

### ⏰ 定时任务可视化编辑器
- 5 字段（分/时/日/月/周）+ 命令输入
- 实时 cron 表达式预览
- 查看当前 crontab，一键保存追加新任务

### 🔗 管道命令构建器
- 多步骤命令管道构建，至少 1 步，可删除步骤
- 实时预览完整管道命令
- 一键执行

### 💻 Docker 容器内执行命令
- 容器列表中增加执行按钮
- 弹出命令输入框，在容器内执行任意命令并显示结果

### 📦 检查更新
- 启动时自动检查 GitHub Release 最新版本
- 有新版本时在侧边栏显示提示

## v1.0.0 功能

### 📖 命令速查
- **130+ 常用 Linux 命令**，按 13 个分类浏览
- **难度分级**：⭐入门 / ⭐⭐进阶 / ⭐⭐⭐高级
- **中文搜索**：输入"查看进程"、"安装软件"等中文也能找到命令；搜索结果关键字高亮
- **场景引导**：每个命令标注典型使用场景
- **关联推荐**：查看一个命令时推荐相关命令
- **一键分享**：点击分享按钮复制命令完整信息
- **悬停详解**：鼠标悬停在示例上查看参数解释

### 🌱 新手入门
- **14 天学习计划**：从文件管理到 Shell 编程，每日主题 + 推荐命令
- **完成追踪**：学习过的命令自动标记进度

### ⚡ 命令执行
- 内置终端模拟器，一键执行命令
- **超时保护**（60 秒自动停止）
- **取消按钮**：可随时取消正在执行的命令
- **输出高亮**：错误红色、警告黄色、成功绿色
- **执行耗时**：每次执行显示耗时秒数及成功/失败状态
- **复制输出**：一键复制执行结果
- **危险命令警告**：自动检测并二次确认 `rm -rf` 等破坏性命令
- 执行历史记录（含时间戳，支持方向键切换，一键清空）

### 🧭 分步引导
提供 dd、rsync、tar、ssh、fdisk、chmod、find、grep、sed、docker 共 10 个复杂命令的分步引导：
- 填写参数表单，实时预览命令
- 一键执行构建完成的命令

### 🔧 报错智能排查
自动识别 20+ 种常见错误并提供中文解决方案：
- `command not found` → 提示安装
- `Permission denied` → 检查权限和 sudo
- `Connection refused` → 检查服务和防火墙
- 端口占用、磁盘满、包冲突等

### 🖥️ 系统信息
- 操作系统、内核、CPU、内存、磁盘、GPU 信息
- **发行版识别**：自动检测当前 Linux 发行版并显示在侧边栏
- **发行版筛选**：按发行版过滤兼容命令
- **平台标签**：命令卡片显示各发行版的兼容性
- 实时监控（CPU / 内存 / 磁盘，3 秒刷新）

### 🐳 Docker 管理
- 列出所有容器（含状态、端口映射）
- 启动 / 停止 / 重启容器
- 查看容器日志

### 🔗 SSH 管理
- 添加 / 编辑 / 删除 SSH 连接
- 一键测试连接

### 📦 安装指南
- AI 工具（Ollama、CUDA、PyTorch 等）
- 开发环境（Miniconda、Node.js、Go、Docker 等）
- 常见问题（pip 慢、端口占用、磁盘空间不足等）
- **一键执行**：指南中的命令可直接点击运行

### 📄 包管理速查
**跨发行版包管理器对比表**（apt / yum / dnf / pacman / zypper）：
- 安装、搜索、更新、卸载、清理等 9 种操作
- 点击任意单元格直接执行对应命令

### ⭐ 收藏 & 笔记
- 收藏常用命令，侧边栏显示收藏数量
- 一键清空所有收藏
- **导出为 PDF**：将收藏命令生成为速查表小抄（支持打印）

### 🔍 搜索历史
- 自动保存最近 10 条搜索记录
- 搜索框聚焦时显示历史，点击即搜

### ⌨️ 全局快捷键
- `Ctrl+Shift+L`：聚焦搜索框并全选文本
- `Ctrl+F`：聚焦搜索框

### 🎨 自定义
- 深色 / 浅色主题
- 12 种强调色
- 3 种字体大小
- 自定义命令别名
- 数据导出（JSON）

## 技术栈

| 层 | 技术 |
|---|---|
| 桌面框架 | [Wails v2](https://wails.io/) |
| 后端 | Go 1.23 |
| 前端 | 原生 JavaScript + Vite |
| 样式 | 纯 CSS 变量 |

## 构建

```bash
# 开发模式
wails dev

# 构建
wails build
```

## 项目结构

```
├── app.go              # Go 后端（命令数据、系统接口、Docker/SSH/引导数据）
├── main.go             # Wails 入口、窗口配置
├── frontend/
│   ├── index.html      # 主页面（含所有 overlay 弹窗）
│   ├── src/
│   │   ├── main.js     # 前端逻辑（渲染、事件、状态管理）
│   │   └── style.css   # 样式表（CSS 变量主题系统）
│   └── dist/           # 构建产物
└── build/
    ├── bin/            # 编译后的可执行文件
    └── package/        # 打包配置
```
